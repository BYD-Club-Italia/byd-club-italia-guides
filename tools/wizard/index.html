Come caricare il pacchetto su GitHub

Il branch main è protetto: tutte le modifiche passano da una Pull Request. Scegli come procedere:

  1. Vai su guides/ nel repository, clicca Add file → Upload files, trascina <slug>.md.
  2. In fondo alla pagina, lascia selezionata la checkbox «Create a new branch for this commit and start a pull request». Suggeriamo come nome di branch guida/<slug>.
  3. Premi Propose changes e poi compila la PR. Suggerimenti:
    • Titolo PR: Aggiunta guida: <titolo>
    • Descrizione: Nuova guida creata col wizard.
  4. Sulla branch della PR appena creata, vai su images/, crea la cartella <slug>/ (digitando <slug>/dummy come filename quando crei un nuovo file e cancellando dummy) e fai Add file → Upload files con tutte le immagini contenute nello zip.
  5. Aspetta review e merge. Il sito si rigenera automaticamente.
  1. Crea un nuovo branch: git checkout -b guida/<slug>
  2. Estrai lo zip nella root del repo (sovrascrive solo i file della tua nuova guida).
  3. Stage e commit: 
    git add guides/<slug>.md images/<slug>/
git commit -m "Aggiunta guida: <titolo>"
  4. Push e apri la PR dal link che git push stampa nel terminale: 
    git push -u origin guida/<slug>
